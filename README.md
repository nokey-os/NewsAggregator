# Новостной агрегатор с лентой в сообществе vk.com
https://nokey.pythonanywhere.com/

## Описание проекта

Проект представляет собой интеллектуальный новостной агрегатор, который позволяет отслеживать, фильтровать и публиковать новости непосредственно в сообщество ВКонтакте. Система ориентирована на удобный поиск информации по ключевым словам и быструю публикацию отобранных материалов в социальной сети.

## Основной функционал

### 1. Поиск новостей
- **Поиск по ключевым словам** – пользователь может вводить ключевые слова или фразы, а так же стоп-слова для фильтрации новостей.
- Поиск осуществляется по заголовкам и содержимому новостей.
- Результаты отображаются в виде ленты (главная панель).

### 2. Просмотр новостей
- **Модальное окно** – при клике на новость открывается всплывающее окно с полным текстом материала.
- **Переход к первоисточнику** – из модального окна доступна прямая ссылка на оригинальную статью.
- Удобная навигация между новостями без перезагрузки страницы.

### 3. Отбор и публикация в VK.com
- **Выбор новостей** – пользователь может отметить одну или несколько новостей для публикации.
- **Выбрать все новости** – функция быстрого выделения всех новостей из текущего списка.
- **Публикация в сообществе VK** – выбранные новости автоматически формируются в пост или серию постов и отправляются в указанное сообщество ВКонтакте.
- Поддержка форматирования и ссылок на первоисточники при публикации.

## Интерфейс и пользовательский опыт

### Главная панель
- Расположена в верхней части экрана.
- Содержит поле поиска по ключевым словам.
- Выводит список новостей, соответствующих запросу.

### Модальное окно просмотра
- Отображает полный текст новости.
- Содержит активную кнопку/ссылку на первоисточник.
- Закрывается по клику вне окна или на крестик.

### Панель управления публикацией
- Кнопки: «Опубликовать выбранное», «Выбрать все новости».
- Интеграция с API ВКонтакте для отправки постов в сообщество.
- Отображение статуса публикации (успешно/ошибка).

## Заключение

Новостной агрегатор с лентой в сообществе VK.com – это готовое решение для эффективного поиска, отбора и публикации новостей прямо в социальную сеть без ручного копирования ссылок и текста. Проект сокращает время на подготовку контента и повышает регулярность постов в сообществе.
